Create purpose

Description

Create purpose is a non-profit organization that helps orphanages in Tijuana by providing psychological help, education, and activities. Some of the projects are coding classes for kids, giving them tools to explore the area of coding at a young age. Another project is agriculture-based learning where the kids learn how to grow food in a sustainable way, using the land and tools from the environment.

My contribution 

Agriculture Technologies

Working with the non-profit to research and develop a sustainable solution to create an agricultural system using technology to provide food for orphanages while requiring little human intervention. Currently implementing a hydroponic solution like a previous project, I develop to see the feasibility and scalability.
